# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-15

### Added
- Initial release of QueryConsole
- Read-only SQL query execution interface
- Environment-based access gating
- Configurable authorization hooks
- SQL validation and security enforcement
- Automatic row limiting
- Query timeout protection
- Client-side query history with localStorage
- Comprehensive audit logging
- Modern responsive UI
- RSpec test suite
- Generator for easy installation

### Security
- Multiple layers of security validation
- Keyword-based SQL injection prevention
- Multiple statement blocking
- Resource exhaustion protection
